Phold, Pholda, Pholds, Pholdas time based sample-and-hold pattern

Part of BenoitLib: http://github.com/cappelnord/BenoitLib


superclass: FilterPattern


Phold, Pholda(dur, pattern, [clock = TempoClock.default])

Pholds, Pholdas(dur, pattern)


repeat each element for a specific duration. Pholds/Pholdas uses seconds instead of beats. Phold/Pholds uses the current time/beat as a reference point for the retreival of the next new value. Pholda/Pholdas uses the last point in time. Therefore effects as undersampling or aliasing can occur with Pholda/Pholdas.


dur may be a pattern, so the durations can vary each iteration.


Phold is actually very simillar to Pstep.


// examples



// Phold used for harmonic progression

(

x = Pbind(

\degree, Pseq([0,2,4,7],inf),

\dur, Pseq([4,2,1],inf) / 8,

\mtranspose, Phold(4, Pseq([0,2],inf)),

\scale, Phold(8, Pseq([Scale.minor, Scale.major],inf))

).play;

)


// Phold doesn't allow aliasing/undersampling

(

x = Pbind(

\degree, Phold(1/16, Pseq([0,2,4,7],inf)),

\dur, 1/8

).play;

)


// Pholda drops every second value

(

x = Pbind(

\degree, Pholda(1/16, Pseq([0,2,4,7],inf)),

\dur, 1/8

).play;

)


// now it's getting funky :-)

(

x = Pbind(

\degree, Pholda(1/15, Pseq([0,2,4,7],inf)),

\dur, 1/8

).play;

)


// also works this way

(

x = Pbind(

\degree, Pholda(1/7, Pseq([0,2,4,7],inf)),

\dur, 1/8

).play;

)


// random sample-and-hold

(

x = Pbind(

\degree, Phold(Pseq([4,2,1],inf) / 2,Prand((0..13),inf)),

\dur, 1/8

).play;

)
