Plast remembers the last value of a key

Part of BenoitLib: http://github.com/cappelnord/BenoitLib


superclass: Pattern


Plast(key)


remembers the last value of a given key. If this is the first event in a stream the current value of the key is specified. Please note: Plast must be used after the definition of the key (and after keys, which are important to the key. For example if you specify degree, then ask for freq and THEN the scale Plast will return a wrong value.Â

SynthDef(\slide, {|out=0, freq=440, lfreq=440, amp=0.1, gate=1, pan=0, slide=0.05|

var sig = LFPulse.ar(Line.kr(lfreq,freq,slide));

var env = EnvGen.ar(Env.adsr(0.01,0.1,0.4,0.1), gate, doneAction:2);

sig = RLPF.ar(sig, XLine.kr(3000,1000,0.15),0.11);

sig = Pan2.ar(sig, pan) * env;

Out.ar(out, sig * amp);

}).add;

)
